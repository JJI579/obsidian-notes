What is an array?

numbered collection of data of the same type each cell has index, and it starts on 1, not zero-indexed.

can use a loop to access elements of the array

sum <- 0;
i <- 1;
while i <= n do
begin
    sum <- sum + A[i]
    i <- i+1
end
output sum

Search Methods

Linear

Iterates and performs a comparison between each element in the array and \(x\) sequentially. can be performed on ordered and unordered lists, however if ordered suggested to use alternative search method.

Time Complexity

BEST: O(1) - first element in the list WORST: O(\(n\)) - as it would iterate through all values of the list / is the last number in the list ## Binary half list everytime and see if number (\(x\)) is > than element in halfed index Only used with sorted list.

Time Complexity

BEST: O(1) = the element was the number in the middle of init instruction WORST: O(\(\log n\)) = required to perform the action
