Pairwise Independent

\(A_{1} \dots A_{n} \text{ are pairwise independent if every pair of events are independent: for all distinct k,m}\)

\(P(A_{m} ∩ A_{k} ) = P(A_{{m}})P(A_{{k}} )\)

if the intersection is always the same then it is along the right lines but to confirm it is pairwise independent, you need to multiple the probability of A1 with probability of A2 and it should return the same value as the intersection

forget about this apparently we only care about mutual independence ### Mutual Independent If every event is independent of any intersection of the other events

\(P(A_{{k1}} ) × · · · × P(A_{km} ) = P(A_{{k1}} ∩ · · · ∩ A_{km} )\)

Performing the intersection of all of them together should be the same as multiplying the probability of all of them together. if the same then it is mutual independence

Bayes Theorem

if \(P(A) > 0\) then

\[ P(B|A) = \left(\frac{{P(A|B) *P(B)}}{P(A)}\right) \]

With multiplication rules this can be changed to have the formula of anything else

Application of this

S = stiff neck because of meningitis (50% from common knowledge) M = \(\frac{1}{50000}\) S = stiff neck in general is \(\frac{1}{20}\)

\(P(M|S) = \frac{{P(S|M)*P(M)}}{P(S)}\)
