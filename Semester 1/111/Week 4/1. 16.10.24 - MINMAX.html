Intelligent Agent Recap

Sensors - to perceive the environment Deliberation - what action it should take Action - how does it perform it

though intelligent agents can determine their state, it does require some representation of the existing environment and to conclude the action the agent wants to perform requires reasoning

Basic Idea of reasoning

  • Tells the agent what it needs to know
  • Uses reasoning to deduce relevant consequences The idea above is called declarative approach

a Knowledge Base which contains facts and general knowledge of the subject in some formal language a Reasoning engine that provides the relevant consequences from the Knowledge Base

Knowledge Graph

Summary - KR&R

To store knowledge and information to them reasoning and determine an action - this is referred as the declarative approach The languages were first developed in logic but is now the discipline of comp sci

Any language that has some rule-based reasoning or propositional logic, then it is a KR&R

propositional logic - referencing statements in its knowledge base

Rule Based Languages

KR&R uses a formal language, therefore it has a syntax to determine its reasoning from ## Syntax - An individual name denotes an individual object, they are often called constant symbols, pretty much anything that is a name that identifies something LiverpoolFC ArneSlot England - An individual Variable which is a placeholder for an individual name This is denoted as \(x\), \(y\), \(z\), \(x_{1}\) - A class name denotes a set of individual objects, kind of like python, youd have a main class of human and subclasses of a person When not defined, this is denoted as \(A, B, C, A_{1}, A_{2}\)
