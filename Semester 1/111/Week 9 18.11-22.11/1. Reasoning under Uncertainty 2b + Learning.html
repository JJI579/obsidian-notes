What is machine learning

provess of converting experience into knowledge takes input of training data, outputs the knowledge through another program / middleware

Types of Learning

  • Supervised Learning it has labelled data and it generalises the examples into a function, the function \(f\) is called a classifier
  • Unsupervised Learning aims to find structure in data without references / examples
  • Reinforcement Learning learns from rewards and punishments in dynamic setting in which agents have a long-term goal.
