## Random
Assume there is a function that defines / denotes the event of a Probability Space and that the space is finite.
\(F: S\to R\)
Therefore given the random variable F, the set of numbers r derived through the function \(F\) is also finite.
### How is it written?
the event that \(F\) takes the value R, most commonly wrote like \(\{ s | F(s) = r \}\), is now written as (F=r)
therefore in syntax, \((F=r) = \{ s | F(s) = r \}\)
### How to apply it?
It can be used of two ways
applied through direct use, such as \((F = r)\) or in other notation through propositional logic, such as ¬\((F = r)\)

## Conditional Distributions
Prerequisites Have a full joint distribution
Notation P notation for conditional distributions
P(F|G) gives condition distribution of F given that G in the context of F and G being referenced as random variables (r,s), for all r,s
### Example
conditional probability distribution for Cavity | Toothache = 1
\(P(\text{Cavity = 1 | Toothache = 1}) = { \dfrac{\text{P(Cavity = 1, Toothache = 1)}}{P(\text{Toothache = 1})}}\)
P(Cavity = 1, Toothache = 1) = \(P(\text{Cavity} = 1) + P(\text{Toothache} = 1)\)
## How to know when to use what
In the scenario above, since we have a [[Recap for Class Test 2#Joint Probability Distribution]] then it is okay to sum the values of each
However, if we did not have a Full Joint Probability Distribution then you can utilise conditional Probability to identify the values as such
P(Cavity = 1 | Toothache = 1) = \(\dfrac{P(Cavity = 1, Toothache=1)}{P(Toothache = 1)}\)
P(Cavity = 1, Toothache = 1) = \(P(\text{Toothache} = 1 | \text{ Cavity} = 1) * P(Cavity = 1)\)
Through this you can identify the values.

Independence

How to identify independence

They are independent if this follows

P(F,G) = P(F) x P(G)

therefore comparison whether its with a joint probability distribution or utilising the multiplication rule

\(P(\text{Toothache} = 1 | \text{ Cavity} = 1) * P(Cavity = 1)\)

Through independence this provides…

Because of two or more variables being independent, it means that we can identify formulas which are equivalent and therefore can be used.

Example: Weather / Dental

\(P(\text{Weather = sunny | Toothache =} r_{1}, \text{Catch =} r_{2}, \text{Cavity =} r_{3})\) = P(Weather = sunny)

and this equivalates to

P(Weather = sunny, Toothache = \(r_{1}\), Catch = \(r_{2}\), Cavity = \(r_{3}\)) = P(Weather = Sunny)P(Toothache = \(r_{1}\), Catch = \(r_{2}\), Cavity = \(r_{3}\))

With Joint Probability Distributions there also comes around another formula

P(Weather, Toothache, Catch, Cavity) can also equal P(Weather) x P(Toothache, Catch, Cavity) ## Conditional Independence

They are conditionally independent if the formula below follows Pasted image 20241127113207.png

Furthermore, rather than looking at 2 variables just like [[Recap for Class Test 2#Pairwise Independent|Pairwise Independency]] and [[Recap for Class Test 2#Mutual Independent|Mutual Independency]], Conditional Independence looks at 3 variables.

Equivalence in Conditional Independence

P(\(a_{1}, a_{2} | a_{3}\)) = \(P(a_{1}|a_{3}) *P(a_{2}|a_{3})\) is equivalent to \(P(a_{1}|a_{3}) = P(a_{1}| a_{2}, a_{3})\) (because a2 is independent to a1) ## What can we do with conditional independence?

We can compute the Joint Probability Distribution through only using the probability distributions

how do we compute this? utilise the multiplication rule P(\(a_{1},a_{2},a_{3}\)) = \(P(a_{1},a_{2}|a_{3})\) * P(\(a_{3}\))

then use conditional independence on this

\(P(a_{1} | a_{3}) * P(a_{2} | a_{3}) * P(a_{3})\)

and therefore you can now compute joint probability distribution with the assumption that you’ll have the data according to a1, a2, a3

Expected Values

What is expected value?

Consider a random variable F with values from real numbers \(x_{1},\dots,x_{n}\)

Assume that P(F=\(x_{i}\)) = \(\dfrac{1}{n}\)

From this, the expected value E[F] is equal to It is the average of all possible values \(x_{1},\dots,x_{n}\) of F

If the probabilities P(F = xi ) are not all equal, we take the probability-weighted average.
