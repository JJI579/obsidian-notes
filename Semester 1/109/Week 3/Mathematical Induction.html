proving by induction that a property holds for every natural number \(n\)

  • Prove that the property holds for some initial value (eg: n=0)
  • Prove that if the property holds for \(m\) (for any natural number \(m\)) then itll hold for \((m+1)\)

Prove that for all integers \(n\geq 1\), \(5^n-1\) is divisible by 4

BC : 1

\(5*1 = 5 - 1 = 4\) Therefore true

therefore we can interpret \(5^n-1 = 4k\)

IS: now we do it for \(n+1\) therefore we can interpret it as

WRONG \(5^{n+1}-1\) = \(5 * 4k\) \(5 * 4k = 20k\) \(20k = 4(5k)\)

Correct \(5^{k+1} -1 = 5*5^k-1\) Can be written as \(5^{k+1} -1 = (4+1)*5^k-1\)

\(5^{k+1} -1 = 4 * 5^k *5^k -1\)
