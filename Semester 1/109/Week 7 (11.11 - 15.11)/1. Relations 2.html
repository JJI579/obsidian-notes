Recap on subset Notation

#subset-notation

There are two types of subsets: Subset \(\subseteq\) Proper Subset \(\subset\)

a proper subset is when the set A fits into set B without being the distinct amount of elements between each set

a normal subset is the same however it can be the distinct amount of elements

Examples

set A = \(\{1,2,3\}\) set B = \(\{1,2,3,4\}\) set C = \(\{1,2,3\}\)

Proper Subset \(A \subset B\) it has the same elements and B has more elements that A itself

not proper subset \(A \subseteq C\)

as it has the same amount of elements

However be aware, proper subsets can also be represented with \(\subset\) notation as it also fits.

Cartesian Product

#cartesian-product

\(A*B = \text{\{(a,b\} | a }\in A\) and \(b \in B\)\(\}\) aka, the sets of both A and B are merged together in every possible combination \(A = \{1,2,3\}\) \(B = \{4,5,6\}\) (1,4) (1,5) (1,6) (2,4) (2,5) (2,6) (3,4) (3,5) (3,6)

Binary Relation

#binary-relation binary relation \(R\) between two sets A and B is \(R ⊆ A*B\) which can be represented by a digraph

Inverse

Inverse relation \(R^{-1} = \{\)\((b,a) | (a,b) \in R\}\)

\[ \begin{align} R = \{(1, 2), (3, 4), (5, 6)\} \\ R^{-1} = \{(2, 1), (4, 3), (6, 5)\} \end{align} \]

Composition

#composition S ◦ R = {(a, c) | exists b ∈ B such that aRb and bSc}

aRb means set A is directly related to B bSc means set B is directly related to C however the actual result (a,c) means that set A and set C are indirectly related!

\(A=\{1,2\}\) \(B = \{3,4\}\) \(C=\{5,6\}\)

relation between A and B \((1,3),(2,4)\)

relation between B and C \((3,5), (4,6)\)

therefore through indirect relation (a,c) \((1,5), (2,6)\)
