Properties of Binary Relations

#binary-relation

∀ # Function functional relation defined on all elements.

Reflexive

Reflexive is when xRx for all \(x \in A\) \(∀x \text{ }A(x) \to xRx\)

this is every element related to itself \(A= \{1,2,3\}\) Relation R: R = \(\{(1,1), (2,2)\dots\}\)

Symmetric

Symmetric is when xRy impliex yRx for all x,y \(\in\) A \(∀ x,y \text{ } xRy\to yRx\)

this suggests that when x,y are related, then y,x will exist in A also

\(A =\{1,2,3\}\) \(\text{Relation R: R=} \{(1,2), (2,1)\}\) since (1,2) \(\in A\) therefore (2,1) \(\in A\)

Antisymmetric

Antisymmetric is when xRy and yRx implies x = y for all \(x,y \in A\) \(∀ x,y, xRy \text{ and } yRx \to y=x\)

this means there is one pair and there cannot be the reverse eg: let A = {1,2,3} Relation R: \(R = \{ (1,2), (1,3), (1,1) \}\) Relation R is is antisymmetric as \((1,3)\) \(\in R\) and (3,1) \(\not\in R\)

Transitive

when xRy and yRz implies xRz for all \(x,y,z \in A\) \(∀x,y,z,xRy,yRz \to xRz\)

\(A =\{1,2,3\}\) \(\text{Relation R: R=} \{(1,2), (2,3), (1,3)\}\)

Transitive Closure

#transitive

Given a binary relation on a set A, the transitive closure R* of R is the unqiuely determined relation on A with the following properties - R* is transitive - R \(\subset R*\) - if S is a transitive relation on A and \(R \subseteq S \text{ then } R* \subseteq S\)

Therefore it is technically finishing the Relation to make it so the full set is [[2. Relations 3#Transitive|Transitive]]

Example

Let A = {1, 2, 3}. Find the transitive closure of R = {(1, 1), (1, 2), (1, 3), (2, 3), (3, 1)}

Therefore to close off the transitive relation

it is (3,2), (2,1) (3,3), (2,2) because of this, the relation R is now solved to have the minimum amount of elements for it to be transitive. (where xRy and yRz)
