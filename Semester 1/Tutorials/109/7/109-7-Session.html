suggest n = 2Z? \(f(n) = m\) since we are trying to fit it into both sets we can suggest m is \(25k\)

\(m = 25k\) \(n=2k\)

substitute values in

\(f(n) = 25 * \left( \frac{n}{2} \right)\) \(f(n) = 25 * \left( \frac{2k}{2} \right)\) now split \(\frac{2k}{2} = k\) \(25*k\) \(= 25k\)

therefore n (\(2k\)) = m (\(25k\))
