#flashcards/124

What does the ALU (Arithmetic Logic Unit) do?::Performs bit manipulation and numeric operations.

What does the Control Unit do?::Supplies the ALU with data and tells it what to do.

What are the roles of the MBR and MAR?::The MBR (Memory Buffer Register) and MAR (Memory Address Register) are used by the CPU to interface with main memory. These cannot be accessed by programmers.

What does the Instruction Register do?::Holds the current instruction in memory.

Why are General Purpose Registers used?::Registers provide much faster access than RAM. They can temporarily store data to speed up operations.
Registers have individual names and can be used generally or for specific purposes.

What does the Instruction Pointer (Program Counter) do?::It always holds the address of the next instruction in memory.

What is the first step in the Fetch Execute Cycle?::Copy the address in the Instruction Pointer into the MAR and issue a read request to the MMU.

What happens to the Instruction Pointer after the first step?::The Instruction Pointer is incremented to the next instruction from the MAR.

What happens to the instruction after it arrives into the MBR?::The instruction is copied to the Instruction Register.

What happens after the instruction is copied into the Instruction Register?::The Instruction Register is decoded to determine the instruction.

What happens after the instruction is decoded?::The system fetches any operands or extra data and carries out the instruction via the ALU. Then, the process loops back to step one.

Why does memory usage increase in the Fetch Execute Cycle?::Memory usage increases when fetching additional operands or extra data from memory.

What are instruction sets and what do they do?::Instruction sets are what the CPU uses to execute operations.
They include:
Transfer: Move data between registers.
Arithmetic: Basic math.
Logic: Bit manipulations.
Test: Compares data values and status flags.
Control: Jumps or subroutine calls.
Misc: Helper operations that don’t fit into other categories.

What are opcodes and operands in instruction sets?::
Opcodes: Instructions the CPU translates to perform operations.
Operands: Additional data from registers or main memory needed for the operation.

What are the different addressing modes?:: Immediate: The value is the mode’s value itself (e.g., a number).
Register: Operand value is stored in a register.
Direct: Operand value is in main memory, specified in the mode.
Register Indirect: Specifies a register that acts as a pointer to the operand in main memory.
Implicit: The instruction doesn’t need an operand because it always performs the same operation.
